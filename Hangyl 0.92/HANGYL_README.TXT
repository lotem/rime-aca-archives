hangyl 0.92
hangyl_hanja 0.51

hangyl 0.92, hangyl_hanja 0.51版更新：
1.添加30万餘條韓語詞彙（由瑾昀提供，源自網絡）
2.修正hangyl_hanja詞庫中若干bug
3.附帶壁紙字體改爲Rangya Sans Classic

hangyl 0.9, hangyl_hanja 0.5版更新：
1. hangyl變更爲默認純韓文輸入，並修正若干提示碼問題(如將q/x顯示爲ng）
2. 修正hangyl_hanja.dict中若干錯誤
3. 半角標點有原有的全角中日標點改爲西文標點
4. "$"符號列表中加入韓元符號₩
5. 原附圖改爲壁紙形式

過去版本全部功能如下：
1. 支持模糊音輸入，r=l，f=p，v=b，z=j，x=q等
2. hangyl方案反査可輸入漢字（需先啓用hangyl_hanja方案）
3. hangyl_hanja方案（實際爲中文輸入法），輔助hangyl方案使用，以現代韓國語字音輸入漢字，字數有限，可反査「朙月拼音」

注意：
1. 如需與漢字混合輸入可使用反査功能或到hangyl.schema.yaml中啓用abc_segmenter
2. hangyl方案如需直接上屛，可到hangyl.schema.yaml中engine處註釋掉- fluency_editor；若要enter韓文上屛，可註釋掉- express_editor
